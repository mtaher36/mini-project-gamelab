## Mini Project Gamelab Back End - Express JS REST API

### Installation

1.  Clone this project\
    `git clone https://gitlab.com/adainacara/adainacara-back-end.git`
2.  Cd into your project folder\
    `cd back-end`
3.  Install dependencies\
    `npm install`\
4.  Copy env file\
    `cp .env.example .env`
5.  Start ExpressJS\
    `npm run dev`
